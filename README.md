# PROYECTO DE FÍSICA III
[Enlace al proyecto](https://cmo-utp.vercel.app/index.html)

## Descripción del proyecto
Este proyecto es un clasificador de movimientos oscilatorios utilizando un modelo de aprendizaje automático. El objetivo es identificar el tipo de movimiento oscilatorio (armónico simple, armónico amortiguado) a partir del gráfico de movimiento, y calcular sus parámetros físicos con los datos que se le entregue.
